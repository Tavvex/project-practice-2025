Young&&Yandex

На стенде от Young&&Yandex нам рассказали о наборе в летнюю школу с разными направлениями подготовки:

  1. Школа разработки интерфейсов
  2. Школа бэкенд-разработки
  3. Школа мобильной разработки
  4. Школа аналитиков-разработчиков и другие

Пример кода для алгоритмической задачи "Выставление тегов":

def main():  
    n = int(input())  
    if n == 1:  
        print(1)  
    elif n == 2:  
        print(2)  
    else:  
        tags = [0] * 36  
        tags[1] = 1  
        tags[2] = 1  
        for i in range(3,n+1):  
            tags[i] = tags[i-1] + tags[i-2]  
        print(sum(tags))  
if __name__ == '__main__':  
    main()
